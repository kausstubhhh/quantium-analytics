# Quantium Data Analytics Simulation

This project is based on Quantium’s Data Analytics job simulation on Forage, where real-world retail data was analysed to generate commercial insights and recommendations.

---

## 📊 Project Overview

The project focuses on analysing customer purchasing behaviour, evaluating trial store performance, and delivering data-driven business recommendations.

---

## 🔍 Key Tasks

### Task 1 — Customer Analytics
- Cleaned and explored transaction data  
- Identified key customer segments  
- Analysed sales drivers across segments  

### Task 2 — Uplift Testing
- Selected control stores using similarity metrics  
- Evaluated trial store performance  
- Conducted statistical testing to measure sales uplift  

### Task 3 — Commercial Insights
- Developed a structured business presentation  
- Provided strategic recommendations for category growth  
- Applied Pyramid Principle for executive communication  

---

## 🛠️ Tools Used
- Python (Pandas, NumPy, Matplotlib)
- Google Colab
- PowerPoint (for reporting)

---

## 📈 Key Insights

- Mainstream young families are the primary drivers of chip sales  
- Larger pack sizes are preferred, indicating value-driven behaviour  
- Trial stores (77 & 88) showed statistically significant sales uplift  
- Store-specific factors impact effectiveness of layout changes  

---

## 📌 Recommendations

- Roll out new layout to similar stores  
- Investigate underperforming stores for optimisation  
- Continue testing and monitoring performance
